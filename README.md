The test application on Teradata Covalent + Angular Material

1. Установить Node.js версии выше 6.12.0 (на версии 8.9.4 всё прошло успешно).
2. Выкачать репозиторий.
3. В папке проекта открыть окно команд.
4. Выполнить команду npm install -g @angular/cli
5. Выполнить команду npm install -g yarn
6. Выполнить команду yarn install
7. Выполнить команду npm start
8. После успешной сборки проекта перейти в браузере по адресу http://localhost:4200/
